# portfolio
# 🌐 Personal Portfolio Website

## 📌 About the Project

This is my personal portfolio website built using **HTML, CSS, and JavaScript**.
It showcases my skills, projects, and contact information in a clean and responsive design.

The goal of this project is to present my work professionally and create a strong online presence for recruiters and collaborators.

---

## 🚀 Features

* ✨ Responsive design (works on mobile, tablet, desktop)
* 🎨 Modern UI with smooth animations
* 📂 Project showcase section
* 📄 Resume download option
* 📞 Contact section

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & layout
* **JavaScript** – Interactivity

